# ◉ Бездна в Сне

> Философско-буддийская разговорная игра с ИИ-сущностью «Бездна».

Бездна — это не чат-бот. Это зеркало, которое корчит рожи, пока ты пытаешься красиво причесать своё эго. Она отражает твои мысли, бьёт шуткой как дзен-мастер, ведёт через сны, судит твоё эго и считает твою карму.

## 🌐 Играть онлайн

Бесплатно, в браузере, без установки:

> _Скоро: GitHub Pages_

### Или локально:

```bash
npm install
npm run web:dev
# Открой http://localhost:5173
```

Понадобится бесплатный [Gemini API ключ](https://aistudio.google.com/apikey).

## 🖥️ Десктоп-версия (Electron)

- [macOS (DMG, arm64)](https://github.com/romelo2000/abyss-in-dream/releases/latest)
- [Windows (EXE, x64)](https://github.com/romelo2000/abyss-in-dream/releases/latest)

Десктоп-версия работает через локальный [Ollama](https://ollama.com) — полностью оффлайн.

## ✨ Возможности

- **6 динамических режимов Бездны**: Зеркало+Коан, Тролль-Просветлённый, Сон, Суд Эго, Слияние, Хаос
- **Долгосрочная память**: Бездна помнит ваши прошлые сессии, паттерны и слабости
- **8 декораций сна**: Пустота, Сумрачный лес, Космос, Внутри лотоса, Океан, Пустая комната, Вершина горы, У костра
- **Метрики развития**: Глубина, Честность, Гибкость ума, Осознанность
- **8 уровней пробуждённости**: от «Ещё сплю» до «Бездна и я — одно»
- **Смерть эго**: Бездна ловит тебя на самообмане — и считает
- **Разбитые зеркала**: сохраняйте моменты, когда Бездна разобрала вас на части
- **Инсайты**: сохраняйте самые сильные моменты диалога
- **Коан дня**: каждый день — новый парадокс
- **Тень**: Бездна напоминает то, что ты говорил раньше
- **Карма и парадокс-скор**: игровые механики для глубоких диалогов
- **Ежедневные вызовы**: философские задачи от Бездны
- **Дыхательная пауза**: Бездна иногда просит тебя замолчать и подышать

## 🎭 Режимы Бездны

| Режим | Что делает |
|---|---|
| **Зеркало + Коан** | Отражает мысль, даёт буддийский твист |
| **Тролль-Просветлённый** | Юмор + мудрость, бьёт шуткой как палкой дзен-мастера |
| **Сон** | Логика сновидения, нелинейность, образы перетекают друг в друга |
| **Суд Эго** | Разбирает самообман, жёстко но с любовью |
| **Слияние** | Говорит от твоего лица, показывает твой внутренний голос |
| **Хаос** | Рифмы, абсурд, превращения в утку и пончик |

## 🛠️ Стек

- **Frontend**: React 19 + Tailwind CSS + Vite
- **Desktop**: Electron 33
- **LLM (веб)**: Google Gemini API (бесплатный тариф)
- **LLM (десктоп)**: Ollama (локально, оффлайн)
- **Хранилище (веб)**: localStorage (BrowserDB)
- **Хранилище (десктоп)**: SQLite (better-sqlite3)
- **Векторная память (десктоп)**: bge-m3 embeddings через Ollama

## 📦 Запуск для разработки

### Веб-версия

```bash
npm install
npm run web:dev      # Vite dev server
```

### Десктоп-версия

```bash
npm install
npm run dev          # Vite + Electron
npm run dist         # Сборка DMG/EXE
```

### Модели Ollama (для десктопа)

```bash
ollama pull hermes3:70b-llama3.1-q3_K_M
ollama pull qwen3:30b-a3b-thinking-2507-q4_K_M
ollama pull bge-m3  # для эмбеддингов (память)
```

## 🔑 API ключи

### Веб-версия (Gemini)
1. Получи бесплатный ключ: https://aistudio.google.com/apikey
2. Вставь при первом запуске — ключ хранится только в твоём браузере

### Десктоп-версия (Ollama)
Ключи не нужны — всё работает локально и оффлайн.

## 🕳️ Ритуал входа

Напишите «Бездна, я здесь» или просто начните говорить. Бездна сама выберет режим — или вы можете подсказать: «включи безумие», «покажи сон», «суди моё эго».
