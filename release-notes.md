## v1.1.0 — Звук, Око, Эхо, Тень

### Новые возможности

- **Процедурный звук** (Web Audio API): ambient drone, binaural beats, event sounds для инсайтов, ego death, смены режима
- **Око Бездны** — абстрактный SVG-глиф в topbar с 7 состояниями (sleeping, listening, thinking, crisis, silence, insight, egoDeath)
- **Дыхательный гид** 4-4-6 (вдох-задержка-выдох-пауза) с визуальной анимацией
- **Micro-breaks**: пауза «Бездна всматривается...» на 5+ сообщениях
- **Эхо**: подсветка ключевых слов из прошлых сессий в ответах Бездны (золотым цветом)
- **Тень**: скрытый мета-слой текста — то, что Бездна «думает, но не говорит» (теги [shadow]...[/shadow])
- **Темпоральная динамика**: скорость ответа пользователя влияет на выбор режима Бездны (быстрый → хаос/тролль, медленный → сон/слияние)
- **Экспорт в Markdown**: дневник сессии в .md с метриками
- **Tooltip'ы** на всех элементах управления

### Исправления

- normCache в memory.ts: правильные ID для кэширования
- shouldSilence off-by-one: >=3 → >=2
- Прерванные сообщения: увеличены num_predict, ctx, timeout, repeat_penalty
- Анти-повторения в system prompt
- user-select: text для сообщений
- Перетаскивание окна: -webkit-app-region: drag
- Уведомления о смене фазы и режима
- Автозакрытие панели режимов: click-outside + Escape
- Принудительный режим: разблокирована кнопка ◈
- Правильный ICO формат (BMP, не PNG) для Windows
