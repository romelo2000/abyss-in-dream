Что это?

«Бездна в Сне» — это десктоп-приложение для философских диалогов с локальным ИИ. Вы разговариваете с «Бездной» — сущностью, которая одновременно буддийский мастер-тролль, голос вашего подсознания и lucid dream со встроенным стендап-комиком.

Игра работает полностью офлайн через Ollama. Ваши данные никогда не покидают устройство. Никаких серверов, никакой телеметрии, никаких подписок.

Полная приватность

Всё работает локально. Чат, метрики, память — всё хранится в SQLite на вашем компьютере. ИИ-модель работает через Ollama — локальный сервер, не требующий интернет-соединения.

Установка

1

Установите Ollama

macOS: brew install ollama
Windows: winget install Ollama.Ollama
Или скачайте с ollama.com

2

Скачайте и запустите приложение

Мастер установки предложит выбрать модель и автоматически её загрузит.

3

Начните диалог с Бездной

Выберите сцену сна и погрузитесь в разговор. Бездна ждёт.
